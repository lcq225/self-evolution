## [1.0.0] - 2026-03-29

### Added
- **Auto Error Capture** - Three methods: decorator, context manager, manual
- **Pattern Detection** - Automatic identification of recurring issues
- **AI Attribution Analysis** - 5-Why root cause analysis
- **Smart Consolidation** - Convert learnings to 5 output types (rule, script, skill, tool, card)
- **Evolution Dashboard** - Visual analytics for evolution progress
- **Weekly Review** - Automated periodic reflection
- **Skill Utility Tracking** - Vector-indexed performance monitoring
- **CLI Interface** - Command-line tools for dashboard, review, and status
- **Complete 8-Step Evolution Loop** - From recording to updating

### Changed
- Initial public release
- Comprehensive documentation (README, CONTRIBUTING, CHANGELOG)
- PyPI packaging and distribution setup

### Technical Details
- Python 3.8+ compatibility
- Cross-platform support (Windows, Linux, macOS)
- MemoryCoreClaw integration for persistent storage
- Vector indexing for semantic search
